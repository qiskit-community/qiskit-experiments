An experiment class to measure the amount by which sx and x are not parallel.

Overview

This experiment runs circuits that repeat blocks of sx - sx - y gates inserted in a Ramsey type experiment, i.e. the full gate sequence is thus Ry(Ï€/2) - [sx - sx - y] ^ n - sx where n is varied.

This sequence measures angle errors where the axis of the sx and x rotation are not parallel. A similar experiment is described in Ref.~[1] where the gate sequence x - y is repeated to amplify errors caused by non-orthogonal x and y rotation axes.

One cause of such errors is non-linearity in the microwave mixer used to produce the pulses for the x and sx gates. Typically, these gates are calibrated to have the same duration and so have different pulse amplitudes. Non-linearities in the mixerâ€™s skew can cause the angle to differ for these different pulse amplitudes.

The way the experiment works is that the initial Ry(Ï€/2) puts the qubit close to the \(+X\) state, with a deviation \(Î´Î¸\), due to the misalignment between sx and x (Ry(Ï€/2) is implemented with sx as described below). The first sx - sx do nothing as they should be rotations about the axis the qubit is pointing along. The first y then mirrors the qubit about the \(y\) axis in the \(xy\) plane of the Bloch sphere, so the \(Î´Î¸\) deviation from \(+X\) becomes a \(-Î´Î¸\) from \(-X\). The next sx - sx sequence rotates about the axis that is \(+Î´Î¸\) rotated in the \(xy\) plane from \(+X\), which takes the deviation from \(-X\) from \(-Î´Î¸\) to \(+3 Î´Î¸\). Then the next y mirrors this across the \(y\) axis, taking the state to \(-3 Î´Î¸\) from \(+X\). This pattern continues with each iteration, with the angular deviation in units of \(Î´Î¸\) following the sequence 1, 3, 5, 7, 9, etc. from \(+X\) and \(-X\). The final sx rotation serves mainly to rotate these deviations from \(+X\) and \(-X\) in the \(xy\) plane into deviations out of the \(xy\) plane, so that they appear as a signal in the \(Z\) basis. Because sx has a \(Î´Î¸\) deviation from x, the final sx adds an extra \(Î´Î¸\) to the deviations, so the pattern ends up as 2, 4, 6, 8, etc., meaning that each iteration adds \(2 Î´Î¸\) to the deviation from the equator of the Bloch sphere (with the sign alternating due to the y gates, so the deviations are really -2, 4, -6, 8, etc.).

For the implementation of the circuits, the experiment uses Rz(Ï€/2) - sx - Rz(-Ï€/2) to implement the Ry(Ï€/2) and Rz(Ï€/2) - x - Rz(-Ï€/2) to implement the y. So the experiment makes use of only sx, x, Rz(Ï€/2), and Rz(-Ï€/2) gates. For the experimentâ€™s analysis to be valid, it is important that the sx and x gates are not replaced (such as by a transpiler pass that replaces x with sx - sx), as it is the angle between them which is being inferred. It is assumed that the angle between x and Rz is exactly \(Ï€/2\).
