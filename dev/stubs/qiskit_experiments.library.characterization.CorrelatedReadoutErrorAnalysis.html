CorrelatedReadoutErrorAnalysis#

class CorrelatedReadoutErrorAnalysis[source]#

An analysis to characterize correlated readout error.

Overview

This class generates the full assignment matrix \(A\) characterizing the readout error for the given qubits from the experiment results and returns the resulting CorrelatedReadoutMitigator

\(A\) is a \(2^n\times 2^n\) matrix \(A\) such that \(A_{y,x}\) is the probability to observe \(y\) given the true outcome should be \(x\).

In the experiment, for each \(x\) a circuit is constructed whose expected outcome is \(x\). From the observed results on the circuit, the probability for each \(y\) is determined, and \(A_{y,x}\) is set accordingly.

Note

Updating Results

If analysis is run with replace_results=True then any analysis results and figures in the experiment data will be cleared and replaced with the new analysis results. Saving this experiment data will replace any previously saved data in a database service using the same experiment ID.

If analysis is run with replace_results=False and the experiment data being analyzed has already been saved to a database service, or already contains analysis results or figures, a copy with a unique experiment ID will be returned containing only the new analysis results and figures. This data can then be saved as its own experiment to a database service.
